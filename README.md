📌 Job Board System

A full-stack Job Board Platform built with Node.js, Express, MongoDB, and React (Next.js).
The system connects employers and candidates, allowing job posting, applications, and applicant management.

🚀 Features
👨‍💼 Employer Features
Create / update / delete job posts
View applicants for each job
Manage applicant status:
PENDING
REVIEWED
ACCEPTED
REJECTED
View company job listings
Pagination support for large datasets
👩‍💻 Candidate Features
Browse available jobs
Apply for jobs
Track application status
View applied jobs
🔔 System Features
RESTful API with Swagger documentation
Authentication (JWT)
Role-based access control (Employer / Candidate)
Real-time updates (WebSocket for views)
Populated relational data (User, Job, Application)
🏗️ Tech Stack
Backend
Node.js
Express.js
MongoDB + Mongoose
JWT Authentication
Swagger (API Docs)
WebSocket (Socket.IO)
Frontend
Next.js (React)
TypeScript
Tailwind CSS
Generated API Client (OpenAPI)
